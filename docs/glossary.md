# 용어집

- **Order** [G-01]: 고객이 결제를 완료한 구매 건. "요청"이나 "장바구니"와 다름.
- **OrderLine** [G-02]: Order 안의 개별 상품 항목. SKU + 수량으로 구성. "장바구니 항목"과 다름.
- **Batch** [G-03]: 창고에 입고된 상품 묶음. reference로 식별. "주문"과 다름.
- **Allocation** [G-04]: OrderLine을 특정 Batch에 배정하는 행위. 가용 수량을 감소시킴.
- **Money** [G-05]: 금액 + 통화의 불변 조합. 단순 숫자가 아니라 통화 정보를 포함.
- **SKU** [G-06]: Stock Keeping Unit. 상품 변형의 고유 식별자 (모델+사이즈+색상).
